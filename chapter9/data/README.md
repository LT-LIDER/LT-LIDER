# Data

This directory contains the parallel and monolingual data for the Chapter 9 exercises.
The parallel data are the development (.dev) and test (.devtest) splits from [FLORES-200](https://github.com/facebookresearch/flores/blob/main/flores200/README.md) for English-Catalan and English-Basque.
The monolingual data are splits (20K segments) from the [CCMatrix](https://opus.nlpl.eu/CCMatrix/corpus/version/CCMatrix) corpus for Catalan and Basque.

Please download the .txt files and upload them accordingly to the notebooks.
